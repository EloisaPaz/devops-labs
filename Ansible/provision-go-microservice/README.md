# Ansible - Provisioning Calculator Go with Ansible on Vagrant

## For start use:

    vagrant up

### The calculator will be available in the following url

#### For instructions of use:

- http://55.55.55.150:8080/calc/

#### For additions:

- http://55.55.55.150:8080/calc/sum/{firstNum}/{secondNum}

#### For subtractions:

- http://55.55.55.150:8080/calc/sub/{firstNum}/{secondNum}

#### For multiplications:

- http://55.55.55.150:8080/calc/mul/{firstNum}/{secondNum}

#### For divisions:

- http://55.55.55.150:8080/calc/div/{firstNum}/{secondNum}

#### You can also check the history through the following url:

- http://55.55.55.150:8080/calc/history
